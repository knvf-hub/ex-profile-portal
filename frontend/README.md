# Employee Profile Portal Frontend

Modern employee profile management frontend built with React, TypeScript, Vite, and Tailwind CSS.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
- Zustand
- Axios
- React Select
- Lucide React

---

## Features

- Sign In / Sign Up
- Multi-step registration form
- Cookie-based authentication support
- Profile card preview
- Edit contact information
- Change password
- Upload profile image
- Skill multi-select with custom skill creation
- Profile update history
